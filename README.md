# Spotify Listening Analysis Dashboard

## Project Overview
This Power BI dashboard analyzes Spotify listening history to uncover trends in music streaming behavior.

## Features
- Total Streams
- Listening Hours
- Top Artists
- Top Tracks
- Time Period Analysis
- Completion Status
- Listening Category

## Tools Used
- Power BI
- Power Query
- DAX
- Excel
